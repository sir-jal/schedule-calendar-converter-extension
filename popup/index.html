How to use this extension

Click out or click the button below to close

  1. Go to View Registration Information page (Banner)
    • You could also import a valid .ics file to edit it. Doing this will skip you to step 4.
  2. Click "schedule details"
    • This allows the extension to view your classes.
  3. Click "import schedule"
    • This will load your classes into the extension.
  4. Customize .ics file, then export
    • Unchecking a class excludes the class from the .ics file.
    • Each class has its own options.
    • You may rename a class by double clicking on it or by pressing E or F2 on your keyboard if you're focused on a class.
      • Hit enter or click out of the text box to apply the name.
      • If the textbox is empty, the name will be reset back to the original course name.
    • Unchecking "include waitlisted courses" will exclude all waitlisted courses, regardless of if the course is checked.
    • Some options may overlap each other. For example, say there are 2 options: "include professor name" and "put professor name in title". If the former is disabled, the latter will counsequently be disabled as well.
    • You can mass (de)select by using the appropriate buttons and the select menu.
      • Asynchronous courses will not be affected by these buttons.
  5. If at any point you come across an issue or would like to provide feedback, fill out the feedback form.
  6. If you have any questions, email me.
