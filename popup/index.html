How to use this extension

Click out to close

  1. Go to school registration page (Banner)
  2. Click "schedule details"
  3. Click "import schedule"
  4. Customize .ics file by using the checkboxes and textbox to name it
    • Unchecking a class excludes the class from the .ics file.
    • Each class has its own options.
    • Unchecking "include waitlisted courses" will exclude all waitlisted courses, regardless of if the course is checked.
    • Some options may overlap each other. For example, say there are 2 options: "include professor name" and "put professor name in title". If the former is disabled, the latter will not apply.
  5. Export & Download .ics file
