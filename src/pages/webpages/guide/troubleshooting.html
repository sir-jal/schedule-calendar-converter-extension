Troubleshooting

Intro

This page serves as a guide for troubleshooting issues. Consult this page before creating a ticket.

Guide

Make sure your schedule is fully loaded

Before loading the schedule into the extension, ensure you click on "Schedule Details". Whatever is not displayed on the page will not be detected by the extension.

Courses with no meeting are excluded by default

They will not be included unless the user manually checks it. These courses will also not be affected by bulk settings.

The classes that are loaded may not match the selected term

This especially occurs when you select a different term. If this occurs, usually clicking in and out of "Schedule Details" will resolve the issue. Remember, the extension grabs course information from there.

Refresh the page and try again

Refreshing the page usually clears out issues.

Disable other extensions that affect the webpage

This can include other Banner-related extensions, ad blockers, etc. These could affect the page and therefore interfere with how the extension fetches your course information. Temporarily disable them if applicable.

Use a supported browser

Use a browser that supports chrome extensions, such as Google Chrome and Microsoft Edge.

Ensure the extension has the permission to download files

Without this permission, the extension will not be able to export .ics files.

Clear extension cache

If you are speficially having issues with the school search, try clearing the cache. After clearing it, reopen the extension and try again.

If issues persist, proceed to the feedback hub linked on the home page and submit a ticket.
